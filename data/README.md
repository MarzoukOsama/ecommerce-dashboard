&nbsp;Données



Ce dossier contient les fichiers CSV nécessaires au dashboard.



&nbsp;Fichiers requis



\- `transactions\_cleaned.csv` : Transactions nettoyées

\- `sales\_by\_product.csv` : Ventes agrégées par produit

\- `sales\_by\_month.csv` : Ventes mensuelles

\- `top\_customers.csv` : Top 20 clients

\- `sales\_by\_country.csv` : Ventes par pays



&nbsp;Obtenir les données



Exécutez d'abord le \[projet ETL E-commerce](https://github.com/MarzoukOsama/etl-ecommerce-pipeline) pour générer ces fichiers.



Ou copiez-les depuis `etl-ecommerce-pipeline/data/processed/` vers ce dossier.
